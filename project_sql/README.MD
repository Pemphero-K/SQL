
# Data Analyst Skills & Salary Analysis (SQL Project)

## Introduction

This project explores salary trends and skill demand within data analytics roles using SQL. I started by identifying the top-paying data analytics jobs overall, and then analyzed the skills associated with those roles. The goal was to better understand which skills drive compensation and how demand compares to salary across the market.

## Background

The data analytics field includes a wide range of roles — from traditional data analysts to more technical positions that overlap with data engineering and cloud infrastructure. While tools like SQL and Excel are widely required, there is growing demand for programming and cloud-based technologies.

This project was built to answer a few key questions:

* What are the highest-paying data analytics jobs?
* Which skills are most in demand?
* Which skills are associated with higher salaries?
* Is there a gap between demand and compensation?

## Tools Used

* SQL (data cleaning, aggregation, analysis)
* PostgreSQL / MySQL (database system)
* VS Code (development environment)
* GIT

SQL concepts applied in this project include:

* SELECT, WHERE
* GROUP BY and ORDER BY
* Aggregate functions such as AVG() and COUNT()
* Filtering and sorting
* joins
* Subqueries and CTEs

## Analysis Process

The first query identified the highest-paying data analytics roles overall to establish a salary benchmark.

From there, I analyzed:

1. The most in-demand skills based on job posting frequency
2. The average salary associated with each skill
3. The relationship between demand count and compensation

The analysis revealed that SQL had the highest demand by a large margin, followed by Excel, Python, and Tableau. However, while SQL is foundational, it does not command the highest salary on its own.

Skills such as AWS, Azure, Snowflake, and Spark showed higher average salaries despite lower demand counts. This suggests that more specialized, technical skills — particularly cloud and big data tools — provide stronger earning potential.

The data also showed that office productivity tools (Word, Outlook, Sheets) are common but associated with lower average salaries, reinforcing that they are complementary rather than differentiating skills.

## What I Learned

This project reinforced several important insights:

* SQL is essential for nearly all data analytics roles.
* High demand does not necessarily mean high pay.
* Programming skills like Python increase earning potential.
* Cloud and data engineering tools are strongly tied to higher salaries.
* The line between data analyst and data engineer is becoming less distinct.

On a technical level, I improved my ability to write efficient aggregation queries, structure analysis logically, and extract meaningful insights from real-world data.

## Conclusion

The results show that foundational tools such as SQL and Excel are critical for entering the field, but higher salaries are associated with technical specialization. Analysts who combine SQL with programming and cloud technologies position themselves for stronger compensation and long-term growth.

This project strengthened my SQL skills while also giving me a clearer understanding of the current data analytics job market.
